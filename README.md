-- Projeto DAS--

Projeto academico do curso  DS "Desenvolvimento de Software" no instituto ISTEC "Instituto Superior de Tecnologias Avançadas"
Objetivo: Desenvolvimento de sistema web para a simular, configurar e visualizar a propagação de epidemias


--Tecnologias  utilizadas --

*   **Frontend:** React.js
*   **Backend:** Python (Flask)
*   **Base de Dados:** MySQL
*   **Modelação:** UML (Draw.io)

--Tarefas concluidas (sprint A)--
- [x] Análise e levantamento de requisitos (US_A001)
- [x] Elaboração do Modelo de Domínio (US_A004)
- [x] Configuração do Repositório GitHub (US_A006)
- [x] Planeamento de tarefas - Diagrama de Gantt (US_A010)
- [x] Preparação da Sprint Review e Relatório (US_A009/US_A011)


--Objetivo do sistema--

- Permitir configurar parametros de simulação (ex.: `N`, `I0`, `β`, `γ`, `T`)
- Executar simulações e apresentar resultados (S, I, R ao longo do tempo)
- Calcular e apresentar a metrica **Rt** ao longo do tempo
- Guardar simulações (parâmetros + resultados) em base de dados
- Permitir histórico e consulta de simulações anteriores
- Roles: Utilizador não registado, Utilizador registado e Administrador
